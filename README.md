## Installation Instructions ğŸ› ï¸

Follow these steps to set up Fit Coach on your local environment:

1. Make sure you have **Node.js** installed. You can download it from the official website.
2. Ensure you have **Hardhat** installed globally using the command: 
   ```bash
   npm install --global hardhat
   ```
3. Navigate to the project directory where you have the Fit Coach files.
4. Run the following command to install all dependencies, including Zama FHE libraries:
   ```bash
   npm install
   ```

**Note:** Please do not attempt to `git clone` this repository or use any URLs for setup.

## Build & Run Your Application ğŸš€

Now that you have the project set up, you can compile and run Fit Coach:

1. **Compile the smart contracts:**
   ```bash
   npx hardhat compile
   ```

2. **Run the tests to ensure everything is functioning correctly:**
   ```bash
   npx hardhat test
   ```

3. **Start the application:**
   ```bash
   npm start
   ```

This will fire up your local server, and you can begin exploring Fit Coach in your browser.
